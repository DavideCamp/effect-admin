# Archivio — documenti superati

Questi documenti descrivono il progetto nella sua cornice originale: **la
migrazione di un'app Django esistente**, con DB ereditato, staff esistente e
scadenze di migrazione. Quella cornice è stata abbandonata nella sessione di
design del 2026-07-05: non esiste alcuna app Django, alcun DB, alcuno staff.
Django admin è solo il riferimento di design. Il progetto reale è una
**libreria greenfield** — vedi `../plan.md` e `../roadmap.md`.

Restano qui perché:

- `plan (1).md` e `implementation-plan.md` documentano il PoC, **completato
  con successo il 2026-07-05** (tutti i criteri di successo soddisfatti):
  sono il riferimento storico per il codice del PoC e per le stime reali.
- `roadmap.md` è la vecchia roadmap orientata alla migrazione: superata,
  ma diverse sue parti tecniche (F1 SQL, F2 list, F3 auth, F4 relazioni)
  sono confluite, riviste, nella roadmap nuova.
